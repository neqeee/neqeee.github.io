Substance Designer 製作法陣貼圖

此筆記內容的教學是ArtStation上的Magic Circle Texture，第一章有Substance Designer基礎的節點介紹，推薦給完全沒使用過Substance Designer的初學者看

製作愛心貼圖#

Shape在Transformation 2D移動時邊緣要設定銳利一點，可以在Filtering選擇Nearest

用Transformation 2D調整大小，按住Ctrl + Shift 讓它保持居中及等比例縮放

在愛心外圍添加圓環#

  1. 選取之前新增的Shpae，按Ctrl + D複製一個調整大小。
  2. 連上Edge Dectect，此節點會在對比度區域周圍畫出輪廓。
  3. Shape另外拉出一條線連上Splatter Circular，圓環外側新增三個環繞的圓。
  4. 用Blend把三個形狀（愛心、圓環、外側的3個圓）混合在一起，Blend Mode選擇Max。 這樣基本構圖就完成了。

撲克牌花色#

方法一、用節點組合形狀 方法二、利用Text 節點直接輸入字元 ♥：Alt + 259 ♦：Alt + 260 ♣：Alt + 261 ♠：Alt + 262

可用Alt + 數字鍵輸入，只要確保這個字型有這些符號可以使用就行。

將三種花色輸入到圓環外三個圓中#

在Splatter Circular節點中增加Input Number，將3種花色作為Input輸入進去。

用Level調整Blend混合結果#

將花色跟圓環混合後會有重疊的部份，可以利用Levels節點製作遮罩

用Level取出花色的圓形Mask，Invert之後連接到Blend的Opacity作為遮罩

圓環內增加形狀#

1. 製作 Spark Shape

Shape-Quad Transform GrayScale-Transform 2D（/2 Offset0.5/0.5 ）-Mirror（Invert Axis X: True）-Mirror（Mirror Axis Y）

2. 連接Shape Mapper，調整大小以匹配圓環

被壓縮的閃光形狀有出現分離的圖元，所以需再增加一個形狀來填補之間的空白

3. 圓環的上個節點Shape再連一條Edge Detect出來，用Blend混合

再加上之前做好的愛心，得出一張有不同明度值形狀的貼圖。

利用Edge Detect獲取貼圖邊緣#

放大看可能發現線條間有縫隙，縫隙產生原因是之前使用Transform2D縮放導致黑白邊緣之間有插值，要解決這個問題可以將前面的Mipmap Mode調整成Manual。

合併輪廓跟明度貼圖、增加模糊效果#

  • 使用到的節點：

Shape Glow Grayscale ：增加外光暈

Radial Blur：外圍圓環加一些徑向模糊

在Texture中偽造一些動態模糊以消除特效播放時可能出現的閃頻效果

輸出貼圖#

上面的扳手Icon為Output按鈕。

Output時預設檔名會依照graph_identifier命名，依照個人習慣會將graph命名為T，Output節點的Identifier命名為MagicCircle_A，如此一來會得到檔名為T_MagicCircle_A的貼圖。

下方的Auomatic export when outputs change勾上的話，會在每次Outputs有改變的時候自動輸出貼圖。
